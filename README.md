## EasyDCI

### A Ruby gem to create DCI Applications, easy ;)

## Installation

Add the following line to your Gemfile :

```ruby
gem 'easy-dci', '~> 1.0.0', git: 'https://github.com/jbox-web/easy-dci.git', tag: '1.0.0'
```

then `bundle install`.
